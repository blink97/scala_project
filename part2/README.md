## Usage

* Execute all project

```bash
./master.sh
```

#### Scripts

* Run zookeeper

```bash
./script-zoo-kafka.sh
```

* Run Package sbt

```bash
./script-package.sh
```

* Run Producer

```bash
./script-consumer.sh
```

* Run Consumer

```bash
./script-producer.sh
```


## Requirements

* Scala 2.12.8
* kafka-2.12-2.2.1
* bash
* Java 8
